## 사용법

### 모듈 데이터 취득

1. **스타 레조넌스를 열기** — 게임에 로그인된 상태로 만듭니다
2. **본 툴 실행** — `STModuleManager.exe`를 더블 클릭합니다 (자동으로 관리자 권한이 요청됩니다)
3. **네트워크 모니터링 켜기** — 화면 하단 상태바의 "네트워크 모니터링" 토글을 클릭하여 켭니다
4. **스타 레조넌스 내에서 구역 이동** — 맵 이동이나 텔레포트 등으로 장면을 전환합니다

토글을 켜면 "서버 탐색 중..."이 표시되고, 게임 서버가 발견되면 "서버 연결됨"으로 바뀝니다. 데이터 취득이 불필요할 때는 토글을 꺼 주세요.

모듈 데이터는 구역 이동 시의 통신에서 취득하므로, 새로운 모듈을 획득한 후 데이터를 갱신하고 싶을 때는 본 툴을 실행한 채로 다시 구역 이동을 해 주세요.

### 모듈 목록·정렬

게임 내에서는 지원하지 않는 스탯별 정렬이나 획득 일시 순·합계값 순 등의 정렬을 지원합니다. 필터링과 조합하면 원하는 모듈을 빠르게 찾을 수 있습니다.

※ 획득 일시는 게임 내 데이터가 아니라 본 툴이 모듈을 취득한 일시가 기록됩니다.

### 최적화

1. 화면 상단 탭에서 "최적화" 패널 열기
2. **메인 스탯** (필수) 선택 — +20 도달을 목표로 하는 스탯
3. **서브 스탯** (희망) 선택 — 여유가 있으면 올리고 싶은 스탯
4. 필요에 따라 **제외 스탯** 및 레어리티 필터 설정
5. 실행하면 점수 상위 10개의 조합이 표시됩니다

자주 사용하는 설정은 "패턴 저장"으로 이름을 붙여 저장·불러오기가 가능합니다.

## 최적화 계산 방법

### 개요

보유 모듈 (최대 약 2000개) 중에서 4개를 선택하여 지정 스탯의 **브레이크포인트 도달**과 **+값 합계**를 최대화하는 조합을 구합니다.

### 브레이크포인트

하나의 스탯의 합계값 (4모듈 합산, 최대 +20)에 따라 효과가 단계적으로 상승합니다.

| 합계값 | 효과 |
|--------|------|
| +1 | 발동 |
| +4 | 상승 |
| +8 | 상승 |
| +12 | 상승 |
| +16 | 대폭 상승 |
| +20 | 최대 |

### 점수 계산

각 스탯마다 도달한 브레이크포인트의 **최고 단계**에 따라 점수가 가산됩니다.

| 브레이크포인트 | 메인 (× 1.0) | 서브 (× 0.3) | 미선택 (× 0.15) | 제외 |
|--------------|-------------|-------------|----------------|------|
| +20 도달 | 10,000 pt | 3,000 pt | 1,500 pt | 0 |
| +16 도달 | 5,000 pt | 1,500 pt | 750 pt | 0 |
| +12 도달 | 100 pt | 30 pt | 15 pt | 0 |
| +8 도달 | 50 pt | 15 pt | 7.5 pt | 0 |
| +4 도달 | 20 pt | 6 pt | 3 pt | 0 |
| +1 도달 | 5 pt | 1.5 pt | 0.75 pt | 0 |

이에 더하여 **전 스탯의 +값 합계 × 2**가 점수에 가산됩니다.

> **우선순위:** +20 도달 > +16 도달 > +값 합계 > +12 이하 브레이크포인트
>
> +20 / +16의 점수를 +값 보너스의 이론 최댓값보다 충분히 크게 설정함으로써 브레이크포인트 도달이 항상 우선됩니다.

### 탐색 고속화

C(2000, 4) ≈ 660억 가지를 전탐색하는 것은 현실적이지 않으므로 다음 절차로 필터링합니다.

1. **관련성 필터** — 메인/서브 스탯을 하나도 갖지 않는 모듈 제외
2. **레어리티 필터** — 지정 품질 미만의 모듈 제외
3. **기여도 점수 Top N** — 각 모듈의 기여도 (`Σ(메인값×3) + Σ(서브값×1) + Σ(기타값×0.5)`) 상위 300개로 압축

필터 후 150~300개 정도로 압축되며, Rayon에 의한 **멀티코어 병렬 탐색**과 **가지치기** (2개 선택 시점에서 남은 각 스탯에 +20을 더한 낙관적 상한 점수가 현재 최선을 밑도는 경우 중단)를 조합함으로써 CPU에서 현실적인 시간 내에 탐색을 가능하게 합니다.
